# MediLight Dashboard

Pharmacist-facing React dashboard for the MediLight Dispensing System.

## Setup

```bash
npm install
npm run dev
```

## Connect to Backend

Open `src/MediLight.jsx` and change line 8:

```javascript
const API_URL = "https://YOUR-BACKEND.onrender.com";
```

## Deploy to Vercel

```bash
npx vercel --prod
```

Or connect this GitHub repo to Vercel for auto-deploy.
